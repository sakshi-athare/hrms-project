# 🚀 HRMS - Human Resource Management System

## 📌 Overview

A full-stack HRMS platform to manage employees, attendance, leave, and organizational workflows.

---

## 🧱 Tech Stack

### 🔹 Frontend

* React + TypeScript
* Vite
* React Router
* TanStack Query
* Tailwind CSS

### 🔹 Backend

* Java 21
* Spring Boot
* Spring Security (JWT Authentication)
* Spring Data JPA
* Hibernate ORM

### 🔹 Database

* PostgreSQL (local setup)

---

## 🚀 Key Features

* 🔐 JWT-based Authentication
* 👥 Role-Based Access Control (Admin, HR, Employee)
* 🧑‍💼 Employee Management
* 🗓 Attendance Tracking & Regularization
* 🏖 Leave Management System
* 📊 Dashboard & Reporting
* 🏢 Organization Settings

---

## 📂 Project Structure

/backend → Spring Boot APIs
/frontend → React Application

---

## ▶️ Getting Started

### Backend

cd backend/hrms
mvn spring-boot:run

### Frontend

cd frontend
npm install
npm run dev
